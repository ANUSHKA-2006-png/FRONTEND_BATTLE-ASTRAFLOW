## 📖 Project Overview

AstraFlow AI is a responsive landing page that showcases an AI platform with dynamic pricing, interactive feature sections, and a modern user interface.

The project focuses on clean design, accessibility, responsive layouts, and SEO optimization while maintaining excellent performance.

---

## ✨ Features

- 🎨 Modern responsive UI
- 📱 Mobile-first design
- ⚡ Interactive pricing calculator
- 💰 Monthly & yearly billing toggle
- 🌍 Currency conversion support
- 📊 Regional pricing adjustments
- 🎥 Hero demo video
- 🧩 Bento grid (Desktop)
- 📂 Accordion layout (Mobile)
- 🔍 SEO optimized
- ♿ Semantic HTML structure

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ANUSHKA-2006-png/FRONTEND_BATTLE-ASTRAFLOW.git
```

### Navigate to the project

```bash
cd FRONTEND_BATTLE-ASTRAFLOW
```

### Run locally

Using Python:

```bash
python -m http.server 4173
```

Open your browser:

```
http://127.0.0.1:4173
```
